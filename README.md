## 📖 VaultConnectとは？

VaultConnectは、**プライバシーを最優先**に設計されたSNSプラットフォームです。  
従来のSNSとは異なり、あなたの投稿を誰が見れるかを完全にコントロールできます。

---

## 🔥 機能一覧

### 🔐 プライベートアカウント

| 機能 | 説明 |
|------|------|
| **デフォルト非公開** | 新規アカウントは自動的に非公開に設定されます |
| **フォローリクエスト** | 他のユーザーからフォローされる前に承認できます |
| **公開/非公開切り替え** | いつでもアカウントの公開状態を変更可能 |

**使い方:**
1. サイドバー（モバイルは下のナビ）から「フォロー」をタップ
2. 「公開」/「非公開」ボタンでアカウントの状態を切り替え
3. 非公開時は、フォローリクエストを承認/拒否できます

---

### 👥 コミュニティ

| 機能 | 説明 |
|------|------|
| **コミュニティ作成** | 自分だけのコミュニティを作成できます |
| **公開/非公開コミュニティ** | 誰でも参加可能 or 招待制を選択 |
| **参加/退出** | いつでもコミュニティに参加・退出可能 |
| **メンバー表示** | コミュニティのメンバー数を確認 |

**使い方:**
1. 「コミュニティ」タブを開く
2. 「新規作成」ボタンでコミュニティを作成
3. コミュニティ名、説明、公開設定を入力
4. 他のコミュニティは「参加」ボタンで参加可能

---

### 🎨 テーマカスタマイズ

あなたのUI設定は**Firebaseに保存**されるため、アップデート後も設定が引き継がれます。

| カテゴリ | オプション |
|----------|-----------|
| **外観モード** | ダーク / ライト / 自動 |
| **アクセントカラー** | 18色のプリセット + カスタムカラー |
| **フォント** | システム / Sans / Serif / Mono / Rounded |
| **フォントサイズ** | 極小 / 小 / 中 / 大 / 極大 |
| **カードスタイル** | デフォルト / ボーダー / 浮き出し / グラス / ミニマル |
| **アバター形状** | 丸 / 角丸 / 四角 |
| **背景** | 7種類のグラデーションプリセット |
| **角丸調整** | 0px 〜 24px のスライダー |
| **アニメーション** | なし / 控えめ / フル |

**使い方:**
1. 「テーマ」タブを開く
2. 各カテゴリのタブを切り替えて設定
3. 変更は自動保存されます

---

### 📝 投稿機能

| 機能 | 説明 |
|------|------|
| **暗号化表示** | 投稿は暗号化された状態で表示され、タップで復号化 |
| **リアルタイム更新** | 新しい投稿は自動的に表示されます |
| **プライバシー分析** | AIが投稿内容のプライバシーリスクを分析 |

**使い方:**
1. フィード画面上部の入力欄にメッセージを入力
2. 「投稿」ボタンをタップ
3. 投稿は暗号化状態で表示され、🔒アイコンで復号化可能

---

### 🔒 セキュリティ設定

| 機能 | 説明 |
|------|------|
| **自動復号化** | 投稿を自動的に復号化するかどうか |
| **リスクスキャン必須** | 投稿前にプライバシーリスクをスキャン |
| **ユーザー名マスキング** | フィード内のユーザー名を匿名化 |

---

## 📱 スマートフォン対応

VaultConnectは**スマートフォンに完全対応**しています。

- ✅ **自動リサイズ**: 画面サイズに合わせてUIが自動調整
- ✅ **ボトムナビゲーション**: 親指で操作しやすい下部メニュー
- ✅ **タッチ最適化**: 大きなタップターゲット（44px以上）
- ✅ **セーフエリア対応**: iPhoneのノッチやホームバーに対応

## 🔒 プライバシーへのこだわり

| 従来のSNS | VaultConnect |
|-----------|--------------|
| 投稿はデフォルトで公開 | **デフォルトで非公開** |
| 誰でもプロフィールを見れる | **承認制フォローシステム** |
| 広告のためにデータ収集 | **最小限のデータのみ保存** |
| 設定がリセットされることがある | **設定はFirebaseに永続保存** |

## 💻 開発者向け

### ローカル実行

```bash
npm install
npm run dev
```

### Docker実行

```bash
docker compose up --build
```

### 技術スタック
- **フロントエンド**: React 19 + TypeScript + Tailwind CSS
- **バックエンド**: Firebase (Authentication + Realtime Database)
- **AI**: Google Gemini API（オプション）
