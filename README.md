## 📖 VaultConnectとは？

VaultConnectは、**プライバシーを最優先**に設計されたSNSプラットフォームです。  
従来のSNSとは異なり、あなたの投稿を誰が見れるかを完全にコントロールできます。

### ✨ 主な特徴

| 機能 | 説明 |
|------|------|
| 🔐 **プライベートアカウント** | デフォルトで非公開。承認したフォロワーだけがあなたの投稿を見れます |
| 👥 **コミュニティ機能** | 興味のあるグループを作成・参加して、仲間と交流 |
| 🎨 **UIカスタマイズ** | テーマ、アクセントカラー、フォントなど自分好みにカスタマイズ |
| 🔒 **投稿の暗号化表示** | 投稿をロック/アンロックして表示をコントロール |
| 🤖 **AIプライバシー分析** | 投稿前に個人情報漏洩リスクをAIがチェック |

---

## 🚀 使い方

### 1. ログイン
- **Googleアカウント**でワンクリックログイン
- または**メールアドレス**で新規登録

### 2. プロフィール設定
- 「フォロー管理」から**公開/非公開**を切り替え
- 非公開の場合、フォローリクエストを承認制に

### 3. 投稿する
- ホーム画面で投稿を作成
- 投稿は自動的に暗号化されて表示

### 4. カスタマイズ
「テーマ設定」から自分好みのUIに変更：
- 🌙 ダーク/ライトモード
- 🎨 18色のアクセントカラー
- 📝 5種類のフォント
- 📐 カードスタイル、角丸調整など

## 🔒 プライバシーへのこだわり

| 従来のSNS | VaultConnect |
|-----------|--------------|
| 投稿はデフォルトで公開 | **デフォルトで非公開** |
| 誰でもプロフィールを見れる | **承認制フォローシステム** |
| 広告のためにデータ収集 | **最小限のデータのみ保存** |
| プライバシー設定が複雑 | **シンプルな設定画面** |

## 💻 開発者向け

### ローカル実行

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
```

http://localhost:3000 でアクセス

### Docker実行

```bash
docker compose up --build
```

### 技術スタック
- **フロントエンド**: React 19 + TypeScript + Tailwind CSS
- **バックエンド**: Firebase (Authentication + Realtime Database)
- **AI**: Google Gemini API（オプション）
